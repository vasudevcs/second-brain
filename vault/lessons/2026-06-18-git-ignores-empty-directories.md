# Lesson

Date: 2026-06-18

Project: Second Brain

Concept:
Git ignores empty directories

What I Learned:
Git only tracks files. Empty folders are not committed unless a placeholder file such as .gitkeep exists.

Example:
The PEIS folder structure was not appearing on GitHub until .gitkeep files were added.

Future Use:
Whenever creating project structures with empty folders, add .gitkeep files.
